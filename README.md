### Installation
Run script With Sudo 
```
sudo git clone https://github.com/mrrobot1o1/Ubuntu-setup-Script.git
cd Ubuntu-setup-Script
sudo chmod +x ubuntusetup.sh
sudo ./ubuntusetup.sh
```
